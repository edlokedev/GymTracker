# User Story: 3 - Browse Exercises by Category

**As a** gym user,
**I want** to see a list of exercises organized by muscle groups and exercise types,
**so that** I can easily find and select appropriate exercises for my workout plan.

## Acceptance Criteria

* User can view exercises categorized by muscle groups (e.g., chest, back, legs, shoulders, arms)
* User can view exercises categorized by exercise types (e.g., strength, cardio, flexibility)
* User can navigate between different categories easily
* Each exercise shows its name and category clearly
* User can select an exercise from the list to begin logging sets
* Exercise categories are intuitive and follow common fitness terminology

## Notes

* Categories should be comprehensive and cover major muscle groups
* Consider including exercise descriptions or brief instructions
* Some exercises may target multiple muscle groups - decide on primary categorization
* This provides the exercise selection foundation for workout logging
